# AI in Higher Education – Data Analysis & Dashboard Project

## Overview
This project was developed as part of my MSc dissertation and investigates the role of Artificial Intelligence (AI) in higher education.
The study explores whether AI is perceived as an *enhancer or a disruptor of teaching and learning*, using quantitative analysis and data visualisation techniques.
The project combines statistical analysis, machine learning, and dashboard development to transform raw survey data into meaningful insights for decision-making.

---

## Objectives
- Analyse student perceptions of AI in education using real-world data. 
- Identify key trends in AI adoption, usefulness, and concerns. 
- Explore relationships between knowledge, confidence, usefulness, and risk.
- Apply statistical and machine learning models for prediction.
- Develop an interactive Power BI dashboard to communicate insights.

---

## Dataset
- Source: Kaggle (Survey on Students’ Perceptions of AI in Education)  
- Sample size: 91 students  
- Type: Quantitative survey data (Likert-scale responses).

### Key Variables:
- AI Knowledge (1–10 scale)  
- Feelings / Confidence (1–4 scale)  
- Perceived Usefulness (1–10 scale)  
- Enhancement Score (composite)  
- Risk Score (composite)  
- Classification Label (Enhancing vs Disrupting).

## Methodology

### Data Preparation
- Data cleaning and validation.
- Handling missing values using mean imputation.
- Feature engineering (composite scores for enhancement and risk). 
- Data normalisation for machine learning models. 

### Analysis Techniques
- Descriptive statistics  
- Correlation analysis  
- Multiple regression modelling  
- Machine learning classification  

### Machine Learning Models
- Logistic Regression  
- Decision Tree Classifier  

Models were evaluated using accuracy, precision, recall, and F1-score.

## Key Insights

- Students show *balanced perceptions*, recognising both benefits and risks of AI.  
- Perceived usefulness of AI is high, but confidence and emotional response remain moderate. 
- Enhancement and risk scores are closely aligned, indicating *mixed attitudes*.
- Correlation analysis shows *weak relationships*, suggesting perceptions are complex and multifactorial.  
- Regression models have *low predictive power*, reinforcing that no single factor explains perceptions.  
- Machine learning models achieved moderate performance, with *AI knowledge and emotional factors as key predictors*. 

Overall conclusion:  
AI is perceived as **both an enhancer and a disruptor**, not one or the other.
